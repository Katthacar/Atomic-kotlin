An Abundance of Benefits
The two features we were able to explain here (without requiring more programming
knowledge) make a huge difference whether or not you’re a Java programmer.
If Kotlin is your first language and you end up on a project that needs more
programmers, it is much easier to recruit one of the many existing Java programmers
into Kotlin.
Kotlin has many other benefits, which we cannot explain until you know more about
programming. That’s what the rest of the book is for.
Languages are often selected by passion, not reason…I’m trying to make Kotlin a
language that is loved for a reason.
—Andrey Breslav, Kotlin Lead Language Designer.
