When you chain access to several members using safe calls, the result is
null
if any
intermediate expressions are
null
.
•
[1]
The property
alice.friend
is
null

null

•
[2]
Here, all intermediate calls produce meaningful values.
•
[3]
An Elvis operator after the chain of safe calls provides an alternate value if

null
