A local inner class has access to other elements in the function as well as elements
in the outer-class object, thus
say
,
emit
,
squeak
and
home()
are available within
speak()
.
You can identify an anonymous inner class because it uses an object expression,
which you see in
cat()
. It returns an
object
of a class inherited from
Pet
that
overrides
speak()
. Anonymous inner classes are smaller and more straightforward
and do not create a named class that will only be used in one place. Even more
compact is a
SAM conversion
, as seen in
hamster()
.
