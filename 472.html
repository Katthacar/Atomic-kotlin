When running the tests within IntelliJ IDEA, you may get an error message like “test
events were not received.” This is because IDEA’s default configuration assumes you
will be running your tests externally, using Gradle. To fix it so you can run your tests
inside IDEA, start at the file menu:
“File” -> “Settings” -> “Build, Execution, Deployment” -> “Build Tools” -> “Gradle”
On that page you’ll see a drop-down titled “Run tests using:” which is set to “Gradle
(Default)”. Change this to “IntelliJ IDEA” and your tests will run correctly.
