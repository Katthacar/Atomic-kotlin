Collection
s are a known size, indicated by their
size
property.
Sequence
s are
treated as if they are infinite. Here, we decide how many elements we want using
take()
, followed a terminal operation (
toList()
or
sum()
). A terminal operation
forces the chain of operations to be performed on the
Sequence
.
The standard library also contains an overloaded version of
generateSequence()

element. The lambda must return either the next element or
null

no more elements. In the following example we add user input to a

You can only iterate once through a
Sequence

exception. To make multiple passes through a
Sequence
