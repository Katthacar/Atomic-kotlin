Repetition with
while
A
while
loop continues as long as the controlling
Boolean-expression
produces
true
:
while
(Boolean-expression) {
// Code to be repeated
}
The
Boolean expression
is evaluated once at the beginning of the loop and again
before each further iteration.
// Summary1/While.kt
fun
testCondition
(i: Int) = i <
100
fun
main
() {
var
i=
0
while
(testCondition(i)) {
print(
"."
)
i +=
10
}
}
/* Output:
..........
*/

Boolean
as the result type for
testCondition()
.
