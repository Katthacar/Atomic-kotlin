Kotlin supports class delegation, so instead of writing explicit function implementa-
tions as in the example above, you can simply specify an object to use as a delegate.
To delegate to a class, place the
by
keyword after the interface name, followed by the
member property to use as the delegate:
class
A
(
val
b: BImpl) : B
by
b
Read this as “class
A
implements interface
B
by
using the
b
member object.” The object
you want to use as a delegate must be a constructor argument.
