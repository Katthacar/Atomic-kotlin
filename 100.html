Member functions are called (
invoked
) with the object name, followed by a
.
(dot/period), followed by the function name and parameter list. Here we call the
meow()

Cat

fun

"mrrrow!"

fun

A member function acts on a particular instance of a class. When you call
meow()
,
you must call it with an object. During the call,
meow()
can access other members
of that object.
