add(

(

..
'F'

fun

"[Chars:, a, b, c, d]"
charmap eq
"{A=0, B=1, C=2, D=3, E=4, F=5}"
}

List
and
Map

The Builder Pattern
Hypothetically, you can create constructors to produce all necessary object config-
urations. Sometimes the number of possibilities makes this messy and impractical.
The
Builder
pattern has several benefits:
1. It creates objects in a multi-step process. This is helpful when object construc-
tion is complex.
2. It produces different object variations using the same basic construction code.
