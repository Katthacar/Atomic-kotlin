You can iterate over a range of elements that are whole quantities, like integers and
characters, but not floating-point values.
Square brackets access characters by index. Because we start counting characters
in a
String
at zero,
s[0]
selects the first character of the
String s
. Selecting
s.lastIndex
produces the final index number:
