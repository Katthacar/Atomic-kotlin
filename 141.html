Command-Line Arguments
You can pass a variable number of arguments on the command line, when you
are starting a program. To capture command-line arguments, you must provide a
particular parameter to
main()
:
// Varargs/MainArgs.kt
fun
main
(args: Array<String>) {
for
(a
in
args) {
println(a)
}
}
The parameter is traditionally called
args
(although you can call it anything), and
the type for
args
can only be
Array<String>
(
Array
of
String
).
If you use IDE, you can pass program arguments by editing the corresponding “Run
configuration”. The last exercise in the corresponding course explains how to do that
in IntelliJ IDEA.
