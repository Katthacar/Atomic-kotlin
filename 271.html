names.zip(ids) { ... }
produces a sequence of name-id
Pair
s, and applies a
specified transformation to each
Pair
. The result is a
List
of initialized
Person
objects.
To zip two adjacent elements from a single
List
, use
zipWithNext()
:

The lambda in the second call to
zipWithNext()
specifies an additional transfor-
mation after zipping.
