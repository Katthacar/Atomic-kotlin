Discovering Types in Sequences
When used in a predicate,
is
finds objects of a given type within a sequence:

import

Human(), Human(), Dog(), Alien(), Dog()

main
() {
val

.find { it
is
Dog }
as
Dog?
// [1]
dog
?.
bark() eq
"Yip!"
// [2]
}
Because
