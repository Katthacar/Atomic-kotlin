Member References
You can pass a member reference as a function argument.
Member references
—function references, property references and constructor references—
can replace trivial lambdas that simply call the corresponding function, property or
constructor.
A member reference uses a double colon to separate the class name from the function
or property. Here,
Message::isRead
is a member reference:
