The library function
sortedWith()
sorts a list using a
comparator
, which is an object
used to compare two elements. The library function
compareBy()
builds a compara-
tor based on its parameters, which are a list of predicates. Using
compareBy()
with
a single argument is equivalent to calling
sortedBy()

Function References
Suppose you want to check whether the list contains any important messages, not
just unread messages. You might have a long list of complicated criteria to decide
what “important” means. You can put this logic into a lambda, but that lambda could
easily become large and complex. The code is more understandable if you extract it
into a separate function. In Kotlin you can’t pass a function where a function type is
