Amphibian
looks like a candidate to be an
interface
rather than an
open class
.
However, local interfaces are not allowed.
Local
open
classes should be rare; if you need one, what you’re trying to make is
probably significant enough to create a regular class.
Amphibian
and
Frog
are invisible outside
localClasses()
, so you can’t return
them from the function. To return objects of local classes, you must upcast them to
a class or interface defined
outside
the function:
