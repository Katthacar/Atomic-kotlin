takeIf()
returns the receiver (in this case, the input
String
) if it satisfies the given
predicate, and
null
if the predicate fails (if the
String
is
"XXX"
). Using
takeIf()
,
we don’t need to extract the result of
readLine()
into a separate identifier, as we
did with the
if
expression in
InputLines1.kt
.

takeIf()

<

(predicate(
this
