Note that

are not constructors, but functions. Their

Many collection functions take a predicate and test it against the elements of a
collection, some of which we’ve already seen:
•
filter()
produces a list containing all elements matching the given predicate.
•
any()
returns
true
if at least one element matches the predicate.
•
all()
checks whether all elements match the predicate.
•
none()
checks that no elements match the predicate.
•
find()
and
firstOrNull()
both return the first element matching the predi-
cate, or
null
if no such element was found.
•
lastOrNull()
returns the last element matching the predicate, or
null
.
•
count()
returns the number of elements matching the predicate.
