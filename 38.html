We can be more verbose and specify the type:
val
identifier: Type = initialization
You start with the
val
or
var
keyword, followed by the identifier, a colon, the type,
an
=

val
n=
1
var
p=
1.2
You can say:
val

1
var

1.2
We’ve told Kotlin that
n
is an
Int
and
p
is a
Double
, rather than letting it infer the
type.
