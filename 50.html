String Templates
A
String
template
is a programmatic way of generating a
String
.
If you put a
$
before an identifier name, the
String
template will insert that
identifier’s contents into the
String
:
// StringTemplates/StringTemplates.kt
fun
main
() {
val
answer =
42
println(
"Found $answer!"
)
// [1]
println(
"printing a $1"
)
// [2]
}
/* Output:
Found 42!
printing a $1
*/
•
[1]
$answer
substitutes the value of
answer
.
•
[2]
If what follows the
$
isn’t recognizable as a program identifier, nothing
special happens.
You can also insert values into a
String
using concatenation (
+
):
// StringTemplates/StringConcatenation.kt
fun
main
() {
val
s=
"hi\n"
// \n is a newline character
val
n=
11
val
d=
3.14
println(
"first: "
+s+
"second: "
+
n+
", third: "
+ d)
}
/* Output:
first: hi
second: 11, third: 3.14
*/
