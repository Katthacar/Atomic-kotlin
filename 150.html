class
Contact
(
val
name: String,
val
number: String

override fun

}
fun

() {

"Miffy"
,

)

"Cleo"
,
"098-765-4321"

contacts = mapOf(

Map
s look like simple little databases. They are sometimes called
associative arrays

to a full-featured database, they are nonetheless remarkably useful (and far more
efficient than a database).
