Looping & Ranges
Many programming languages index into an iterable object by stepping through
integers. Kotlin’s
for
allows you to take elements directly from iterable objects like
ranges and
String
s. For example, this
for
selects each character in the
String
"Kotlin"
:
// Summary1/StringIteration.kt
fun
main
() {
for
(c
in
"Kotlin"
){
print(
"$c "
)
// c += 1 // error:
// val cannot be reassigned
}
}
/* Output:
Kotlin
*/
c
can’t be explicitly defined as either a
var
or
val
—Kotlin automatically makes it a
val
and infers its type as
Char
(you can provide the type explicitly, but in practice
this is rarely done).
