Abstract Classes

An interface may declare properties, but the actual data is only stored in subclasses.
An interface isn’t allowed to store values in its properties:
// Abstract/NoStateInInterfaces.kt
package
abstractclasses
interface
IntList
{
val
name: String
// Doesn't compile:
// val list = listOf(0)
}
Both interfaces and abstract classes can contain functions with implementations. You
can call other
abstract
members from such functions:
